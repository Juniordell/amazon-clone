## Building
You must have [Node.js](https://nodejs.org) installed on your computer

```bash
git clone https://github.com/Juniordell/Twitter-Clone
$ cd Twitter-Clone
$ yarn install 
$ yarn start
```

## Technologies
* React JS
* Material UI
* Firebase
* React Router
* React Hooks
